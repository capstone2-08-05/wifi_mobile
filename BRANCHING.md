# Branching Strategy

- `main`: release-ready branch
- `dev`: integration branch
- `feature/<scope>-<name>`: feature branch
- `fix/<scope>-<name>`: bug fix branch

Examples:
- `feature/android-rssi-collector`
- `feature/android-session-upload`
- `fix/android-permission-flow`

Commit prefix:
- `feat: ...`
- `fix: ...`
- `chore: ...`
- `docs: ...`
